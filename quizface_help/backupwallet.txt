backupwallet "destination"

Safely copies current wallet file to destination filename

Arguments:
1. "destination"   (string, required) The destination filename, saved in the directory set by -exportdir option.

Result:
"path"             (string) The full path of the destination file

Examples:
> zcash-cli backupwallet "backupdata"
> curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "backupwallet", "params": ["backupdata"] }' -H 'content-type: text/plain;' http://127.0.0.1:8232/
